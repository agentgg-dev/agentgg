# Samples

Placeholder. Copy-paste starting points for users will live here once
the scanner is implemented — minimally:

- `webapp/` — example `agentgg.config.ts` registering a single project,
  a project-local `agents/` dir, a sample `SECURITY.md`.
- `monorepo/` — multiple projects in one workspace.
- `with-plugin/` — `defineConfig` using a custom plugin to add a
  matcher + a notifier.

Each sample is self-contained: copy the directory next to your real
project, point `root` at your codebase, and run `agentgg scan` from
inside.
